Prototipação de Alta Fidelidade

Criação de fluxos interativos utilizando componentes do Design System estabelecido nas aulas anteriores.

Nota Final 9.5

description Instruções da Tarefa

Nesta atividade, você deve evoluir o wireframe da semana passada para um protótipo funcional de alta fidelidade. O foco principal é a consistência visual e o uso correto dos tokens de design (cores, tipografia e espaçamento).

  • Implementar pelo menos 3 fluxos de usuário completos.
  • Utilizar componentes variantes para estados de botões (hover, active, disabled).
  • Garantir a acessibilidade nas razões de contraste de cores.
  • Exportar o link do protótipo navegável.
